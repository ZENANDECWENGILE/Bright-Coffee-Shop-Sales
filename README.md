# Bright-Coffee-Shop-Sales
Overview

This project analyzes coffee shop sales data to uncover trends in revenue, customer purchasing behavior, product performance, and peak trading periods. The goal was to transform raw transactional data into actionable business insights that can help improve sales performance and customer experience.

Objectives

Identify top-selling products and categories

Analyze monthly and daily sales trends

Determine peak sales hours and low-traffic periods

Measure revenue performance over time

Provide recommendations to increase profitability

Tools Used
SQL
Excel
PowerPoint Presentation

Key Insights

Coffee products generated the highest sales volume

June was the best-performing month, while February had the lowest sales

Morning hours produced the highest revenue

Monday was the strongest sales day, while Saturday underperformed

Recommendations

Launch promotions during low-performing periods

Introduce evening happy-hour offers

Expand best-selling coffee bundles

Increase staffing during peak morning hours

Implement loyalty rewards for repeat customers
